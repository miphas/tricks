
# 快捷键

### 多行编辑  

|      快捷键       |        功能         |
|------------------|--------------------|
|  Alt + 鼠标左键   |   在多个位置插入光标   |
|  Command + D     | 选择多个相同的内容    |


### 行操作

|      快捷键          |        功能         |
|---------------------|--------------------|
|     Alt + Up/Down   |      行上下移动      |
|Alt + Shift + Up/Down|     复制行的内容     |
